About

First and foremost, I love solving real-life problems. And how to solve them, now that we have the advanced machines adaptable as we wish, it's pretty easy to visualize the problems, simulate, test and finally implement the solution. My coding journey began in school in 2014, my first programming language being C. As a computer science major student, I find myself as a slow but steady learner. I enjoyed spending the next 4 years exploring various algorithms, system design and networking.

Fast forward today, I am a graduate student at the Indian Institute of Information Technology, Gwalior pursuing Masters specializing in Computer Networks. I find myself having intermediate level expertise in subjects like Wireless sensor networks and Advanced network technologies. I have project experience in working with microcontrollers NodeMCU & Arduino for technical fests.

My primary choice would be always to program in C & C++. I do code in other languages as well like Python for machine learning and data analytics tasks whereas MATLAB for image processing projects.

My Interests are IoT, Web Development and Machine Learning.

I am currently conducting research on Antenna Design and Optimisation using Machine Learning for sub 6GHz frequency band for 5G IoT applications as part of M.Tech Project.
